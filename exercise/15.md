## 問題 15-2 ★★★ 適応的曲率の設計

「万能の幾何学はない」という本講義の結論を踏まえ、「タスクに応じて曲率を自動選択するモデル」の設計案を提案せよ。

**ヒント:** 曲率をパラメータとして学習可能にする？　Mixed-curveatureの拡張？

**解答:**
**設計案：Adaptive Curvature Embeddings (ACE)**

**基本アイデア:**

- 各トークン（または各層）に対して、曲率パラメータ $\kappa$ を学習
- $\kappa > 0$ ：球面的（等価関係に適する）
- $\kappa = 0$ ：ユークリッド（平坦な関係に適する）
- $\kappa < 0$ ：双曲的（階層関係に適する）

**実装の骨子:**

```15_adaptive_curvature_layer.py
class AdaptiveCurvatureLayer(nn.Module):
    def __init__(self, dim):
        self.kappa = nn.Parameter(torch.zeros(1))  # 学習可能な曲率
        
    def forward(self, x):
        if self.kappa > threshold:
            return spherical_projection(x, self.kappa)
        elif self.kappa < -threshold:
            return hyperbolic_projection(x, -self.kappa)
        else:
            return x  # ユークリッドのまま
```

**予想される困難:**

- 曲率の勾配計算が複雑（多様体上の微分幾何が必要）
- 異なる曲率間の遷移が不連続になりうる
- 学習の安定性

**研究の方向性:** Mixed-curvature spacesの論文を読み、曲率を「離散選択」ではなく「連続パラメータ」として扱う方法を検討する。
