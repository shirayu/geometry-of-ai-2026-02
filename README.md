# 情報幾何学とAIの統一視点

副題: 深層学習の歴史で紐解く、超球面上の「プラネタリウム」構築論

## ショート概要

```txt
深層学習を幾何で統べる全15回講義。
従来のユークリッド空間から、nGPTや双曲多様体が示す「曲がった空間」へのパラダイムシフト。
MoEやTDA等の最新知見を交え、手法の陳腐化に抗う普遍的な「空間の形」を探る。
平坦な地図を捨て、AIの星空を読み解く旅へ。
```
